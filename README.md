# Tourism Web 🌍

A modern static tourism website developed using HTML and CSS.  
This project was built as part of my web development learning journey by implementing the concepts and project guidelines provided by my academy.

## 📖 Overview

Tourism Web is a front-end project that presents tourism destinations through a clean and visually appealing interface. The project focuses on building structured web pages, designing layouts, and improving styling skills using core web technologies.

## ✨ Features

- Clean and simple UI
- Tourism destination sections
- Structured page layout
- Bootstrap-based styling
- Beginner-friendly front-end project
- Static website implementation

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **Bootstrap**
- **Java UI Kit**

## 📂 Project Structure

```bash
tourism-web/
│── index.html
│── style.css
│── images/
```

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Anandu-12233/tourism-web.git
```

### Navigate to the Project Folder

```bash
cd tourism-web
```

### Run the Project

Open the `index.html` file in your browser.
